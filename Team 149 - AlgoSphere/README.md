## Theme: Open Innovation
## Problem Statement
Data Structures and Algorithms are often difficult to understand through theoretical learning. Students struggle to visualize internal operations such as sorting steps, graph traversal, and memory behavior.

---

## Solution
AlgoSphere is an interactive web platform that simplifies DSA using real-time visualizations, hands-on puzzles, and complexity analysis, helping users move from memorization to conceptual understanding.

---

## Features
- Algorithm Visualizer with step-by-step execution  
- Interactive puzzles for applied learning  
- Complexity analysis with growth comparison  
- Custom inputs and speed control  

---

## Data Structures Used

**Arrays**  
Used for implementing and visualizing sorting algorithms such as Bubble Sort, Selection Sort, and Merge Sort. They help demonstrate element comparisons and swaps in a structured format.

**Stacks**  
Implemented using LIFO (Last In First Out) principle. Used in puzzles like reversing elements and understanding function call behavior and DFS logic.

**Queues**  
Used in BFS traversal and simulations. Demonstrates FIFO (First In First Out) behavior, essential for level-order processing in graphs.

**Linked Lists**  
Dynamic data structure used to show node-based memory allocation and pointer manipulation through insertion and deletion operations.

**Trees (Binary Search Trees)**  
Used to demonstrate hierarchical data storage and efficient searching operations like Binary Search and BST insertions.

**Graphs**  
Used for BFS, DFS, and Dijkstra’s Algorithm visualizations. Helps understand traversal, shortest path, and real-world network problems.

**Hash Tables (Conceptual)**  
Included in complexity comparison to explain constant-time operations and efficient data retrieval.
